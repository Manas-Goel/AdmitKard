# AdmitKard

## Frontend

Add a .env file in the frontend root folder<br/>
Add `REACT_APP_BACKEND_URL=THE URL OF THE BACKEND`<br/>
Run `npm install`<br/>
Run `npm start`<br/>

## Backend

Add a .env file in the backend root folder<br/>
Add `DATABASE_URL=THE URL OF THE MongoDB DATABASE`<br/>
Run `npm install`<br/>
Run `npm start`
